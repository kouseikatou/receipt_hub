# 抽出スキーマ詳細

## Gemini Vision への渡し方

`mcp__gemini__analyze_media` はファイルパスまたは URL を受け取る。

- **ローカルファイル**: 絶対パスをそのまま渡す
- **Gmail 添付**: 添付ファイルをいったん `/tmp/` に保存してから渡す
- **Chatwork ファイル**: ダウンロードして `/tmp/` に保存してから渡す
- **メール本文のみ**: テキストを `analyze_media` の `text` パラメータで渡す

## 金額の正規化ルール

Gemini が返す金額をそのまま使わず、以下で正規化する：

- カンマ除去: `1,650` → `1650`
- 通貨記号除去: `¥1,650` → `1650`
- 全角数字変換: `１６５０` → `1650`
- 「円」「円也」の除去

## 日付の正規化ルール

YYYY-MM-DD に統一：

| 入力例 | 変換後 |
|--------|--------|
| 令和6年1月15日 | 2024-01-15 |
| R6.01.15 | 2024-01-15 |
| 2024/1/15 | 2024-01-15 |
| Jan 15, 2024 | 2024-01-15 |
| 令和8年3月15日 | 2026-03-15 |

令和の変換: 西暦 = 2018 + 令和年

## 軽減税率（8%）の判定

以下を含む品目は 8% の可能性が高い（Gemini の判定が null の場合に補完）：
- テイクアウト・お弁当・食材
- 定期購読の新聞

以下は 10%：
- 飲食店での食事（イートイン）
- サービス全般・電子機器・事務用品

## 重複チェック

同一アイテムとみなす条件（2つ以上一致）：
1. 金額が同じ
2. 日付が同じ（±1日）
3. vendor が類似（前方一致、または一方が他方を含む）

## vendor_history.json の構造

```json
{
  "スターバックス渋谷店": {
    "category": "会議費",
    "count": 12,
    "last_confirmed": "2026-03-15"
  },
  "GO タクシー": {
    "category": "旅費交通費",
    "count": 5,
    "last_confirmed": "2026-03-28"
  },
  "AWS Japan": {
    "category": "通信費",
    "count": 8,
    "last_confirmed": "2026-04-01"
  }
}
```

`count >= 3` のベンダーは Gemini 解析をスキップして履歴の category を直接採用する（高速パス）。
`count < 3` のベンダーは Gemini 解析結果を優先しつつ、履歴を参考情報として提示する。
