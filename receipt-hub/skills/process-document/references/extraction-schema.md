# 抽出スキーマ詳細

## PDF テキスト抽出
```bash
# pdftotext が使える場合
pdftotext -layout input.pdf - 

# Python の pdfminer を使う場合
python3 -c "
from pdfminer.high_level import extract_text
text = extract_text('input.pdf')
print(text)
"
```

## 画像 OCR (Gemini Vision)
Gemini の `analyze_media` ツールを使用する。プロンプト例：
```
この領収書・請求書から以下を日本語で抽出してください：
1. 発行日（YYYY-MM-DD形式）
2. 店名・発行元
3. 税込合計金額（数字のみ）
4. 税抜金額（記載がある場合）
5. 消費税率（8%または10%）
6. 書類の種別（領収書/請求書/レシートなど）
7. 品目や用途のメモになりそうな情報

JSON形式で出力してください。
```

## 金額の正規化ルール
- カンマ区切り除去: `1,650` → `1650`
- 通貨記号除去: `¥1,650` → `1650`
- 全角数字変換: `１６５０` → `1650`
- 「円」「円也」の除去

## 日付の正規化ルール
様々な日付フォーマットを `YYYY-MM-DD` に統一:
- `令和6年1月15日` → `2024-01-15`
- `R6.01.15` → `2024-01-15`
- `2024/1/15` → `2024-01-15`
- `Jan 15, 2024` → `2024-01-15`

## 軽減税率（8%）の判定
以下を含む品目は軽減税率8%の可能性が高い:
- 食料品（テイクアウト・お弁当・食材）
- 定期購読の新聞

以下は標準税率10%:
- 外食・飲食店での食事（イートイン）
- サービス全般
- 電子機器・事務用品

## 重複チェックロジック
同一アイテムとみなす条件（2つ以上一致した場合）:
1. 金額が同じ
2. 日付が同じ（±1日）
3. 店名が類似（Levenshtein距離が低い）
