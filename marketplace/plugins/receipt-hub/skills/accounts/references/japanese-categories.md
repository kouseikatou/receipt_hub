# 日本の勘定科目 詳細一覧

## 販売費および一般管理費（フリーランス・個人事業主向け）

### 会議費
- **定義**: 会議・打ち合わせに要した費用（飲食含む）
- **上限**: 1人1回5,000円以下が目安
- **例**: カフェでの打ち合わせ代、会議室のお茶・お菓子代
- **注意**: 飲食代が主目的の場合は交際費になる可能性あり

### 交際費
- **定義**: 取引先・得意先との接待・贈答・懇親のための費用
- **例**: 取引先との会食、手土産、ゴルフ接待
- **注意**: 個人事業主は全額経費計上可能（法人は上限あり）

### 旅費交通費
- **定義**: 業務上の移動に要した費用
- **例**: 電車・バス・タクシー代、飛行機・新幹線代、宿泊費（出張時）
- **ICカード精算**: Suica・PASMO等のチャージは購入時でなく使用時に計上

### 通信費
- **定義**: 業務上の通信に要した費用
- **例**: 携帯電話料金（按分）、インターネット回線料（按分）、切手・宅配便

### 消耗品費
- **定義**: 耐用年数1年未満または取得価額10万円未満の物品
- **例**: 文房具、プリンターインク、USBメモリ、マウス・キーボード、ノートPC（10万円未満）

### 工具器具備品
- **定義**: 取得価額10万円以上20万円未満の器具・備品（一括償却可）
- **例**: PC（10〜20万円）、カメラ、モニター

### 減価償却費
- **定義**: 取得価額20万円以上の資産を耐用年数で分割計上
- **例**: 高額PC、業務用機材、車両（按分）

### 広告宣伝費
- **定義**: 事業の宣伝・広告に要した費用
- **例**: SNS広告費、Google広告、名刺・チラシ制作費、HP制作費

### 外注費（業務委託費）
- **定義**: 外部に業務を委託した費用
- **例**: フリーランスへの制作費支払い、外部ライター費用
- **注意**: 源泉徴収が必要な場合あり（個人への支払い）

### 地代家賃
- **定義**: 事務所・店舗の賃料
- **例**: オフィス家賃、コワーキングスペース利用料、レンタルオフィス代
- **自宅兼用**: 業務使用割合で按分（一般的に20〜50%）

### 新聞図書費
- **定義**: 業務上必要な書籍・新聞・雑誌・デジタルコンテンツ
- **例**: 技術書、ビジネス書、業界誌、Kindle本、オンライン記事定期購読

### 研修費（教育訓練費）
- **定義**: 業務上必要な知識・技術習得のための費用
- **例**: セミナー参加費、オンライン講座（Udemy等）、資格取得費用

### 水道光熱費
- **定義**: 電気・ガス・水道料金
- **自宅兼用**: 業務使用割合で按分（一般的に10〜30%）

### 支払手数料
- **定義**: 各種手数料・サービス利用料
- **例**: 銀行振込手数料、クレジットカード年会費、会計ソフト利用料（freee等）、Stripe手数料

### 雑費
- **定義**: 他の勘定科目に分類できない少額の費用
- **例**: 印鑑購入、慶弔費（少額）
- **注意**: 多用しすぎると税務調査で指摘を受けやすい

## 売上原価
### 仕入高
- 商品・材料の仕入れ

### 外注費（原価）
- 売上に直接対応する外注費（販管費の外注費と区別が必要な場合）

## よく混同される科目の判定フロー

```
飲食代の場合:
  └─ 取引先・顧客あり？
       ├─ Yes → 1人5,000円以下？
       │          ├─ Yes → 会議費
       │          └─ No  → 交際費
       └─ No  → 自分だけ（昼食等）→ 原則経費にならない
                              ↓
                        在宅ワーク中の食事 → 福利厚生費（従業員がいる場合のみ）

ソフトウェア・サービス利用料の場合:
  └─ 年間/月額サブスクリプション？
       ├─ Yes → 支払手数料 または 通信費
       └─ No  → 消耗品費（買い切り）

書籍・コンテンツの場合:
  └─ 業務に直接関係ある？
       ├─ Yes → 新聞図書費
       └─ No  → 経費不可（按分も困難）
```

## 消費税区分
| 区分 | 対象 | 税率 |
|------|------|------|
| 課税（10%）| ほとんどの商品・サービス | 10% |
| 軽減税率（8%）| 食料品（外食除く）、定期購読の新聞 | 8% |
| 非課税 | 土地、住宅家賃、保険料、利子 | 0% |
| 免税 | 輸出取引 | 0% |
| 不課税 | 給与、保険金、損害賠償 | 対象外 |

## インボイス制度（2023年10月〜）
- 適格請求書発行事業者からの請求書でないと仕入税額控除不可
- 領収書・請求書に「T + 13桁登録番号」が記載されているか確認
- 記載がない場合は経過措置期間中は80%控除可能（2026年9月まで）
