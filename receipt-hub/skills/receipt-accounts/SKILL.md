# 日本の勘定科目 知識ベース

詳細な勘定科目一覧・判定ルールは `references/japanese-categories.md` を参照。

## フリーランス・個人事業主によく使う勘定科目（早見表）

| 勘定科目 | 主な用途 | キーワード |
|---------|---------|-----------|
| 会議費 | 打ち合わせの飲食・1人5,000円以下 | スタバ、カフェ、打ち合わせ、会議 |
| 交際費 | 接待・会食・1人5,000円超 | 接待、会食、取引先、夕食 |
| 旅費交通費 | 電車・バス・タクシー・出張 | Suica、交通費、出張、新幹線 |
| 通信費 | 電話・インターネット・郵便 | ドコモ、au、光回線、郵便 |
| 消耗品費 | 10万円未満の備品・事務用品 | 文具、USB、マウス、ノート |
| 広告宣伝費 | 広告・名刺・HP制作 | 広告、名刺、デザイン費 |
| 外注費 | 外部委託・フリーランスへの支払い | 業務委託、外注、制作費 |
| 地代家賃 | 事務所家賃・コワーキング | 家賃、コワーキング、レンタルオフィス |
| 新聞図書費 | 書籍・雑誌・オンライン購読 | 書籍、Amazon Books、Kindle |
| 研修費 | セミナー・勉強会・オンライン講座 | セミナー、Udemy、勉強会 |
| 水道光熱費 | 電気・ガス・水道（按分） | 電気代、東電、中部電力 |
| 福利厚生費 | 従業員向け（個人事業主には不適） | ※個人事業主は使わない |

## 会議費 vs 交際費 の判定基準
- **会議費**: 1人あたり5,000円以下 AND 仕事上の打ち合わせ目的
- **交際費**: 1人あたり5,000円超 OR 接待・もてなし目的

## 家事按分が必要な科目
自宅兼事務所の場合、以下は業務使用割合で按分する:
- 地代家賃（例: 業務30%）
- 水道光熱費（例: 業務20%）
- 通信費（スマホ等、例: 業務50%）
